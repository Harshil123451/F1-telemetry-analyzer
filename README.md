# 🏎️ F1 Telemetry Analyzer

A comprehensive Formula 1 telemetry analysis tool built with Python and Streamlit. Analyze lap times, telemetry data, track maps, and sector times from any F1 race session.

## 🚀 Features

- **Lap Time Analysis**: Compare lap times between drivers
- **Telemetry Visualization**: Speed, throttle, brake, and RPM data
- **Track Mapping**: Interactive track maps with speed visualization
- **Sector Analysis**: Detailed sector time breakdowns
- **Multi-Driver Support**: Compare any two drivers side-by-side
- **Historical Data**: Access data from 2018-present seasons
